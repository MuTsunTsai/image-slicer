# Image Slicer

A simple web-based tool for cropping square regions from images.

## Features

- Open images via file picker or drag-and-drop
- Select square crop sizes: 256×256, 512×512, or 1024×1024
- Drag the selection box to choose the crop area
- Save cropped images using File System Access API (with fallback to download)
- Responsive design for desktop and mobile devices
- Touch-friendly interface

## Usage

Visit the live demo: https://mutsuntsai.github.io/image-slicer/
